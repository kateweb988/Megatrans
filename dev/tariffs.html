  • Дополнительные условия тарификации

    • Основные тарифные категории (для стандартных грузов)

      Стоимость перевозки определяется по фактическому весу или объему — выбирается большее значение.

      Тарифная категория Максимальный вес, кг Максимальный объем, м³ Допустимые габариты (Д x Ш x В), м
      Малый 5 0.1 0.3 x 0.3 x 0.3
      Средний 15 0.1 0.5 x 0.3 x 0.3
      Большой 30 0.1 0.6 x 0.4 x 0.4
      Сверхбольшой 5 0.1 0.3 x 0.3 x 0.3

      Для грузов свыше 50 кг / 0.15 м³ стоимость определяется как большая из двух величин:

      • Стоимость по тарифу "Сверхбольшой".

      • Стоимость межтерминальной перевозки, рассчитываемая исходя из объема или веса (плотность груза = 250 кг/м³).

      Особенности расчета и применяемы поправки

      • Коэффициент укладки: конечный объем груза определяется как фактический объема груза, умноженный на 1,1.

      • Защитная упаковка:

      • — Паллетный евроборт: расчётные параметры умножаются на 1,2

      • — Обрешетка: расчётные параметры умножаются на 1,3; вес груза увеличивается на 25 кг/м³

      Негабаритные грузы

      По весу одного места:

      Вес одного места Наценка
      300–600 кг +15%
      600-1000 кг +30%
      более 1000 кг +50%

      По габаритам:

      Длина стороны Наценка
      3-6 м +25%
      более 6 м +50%

      Примечание: на одно тарное место может быть применен только один процент негабарита
